## 🔑 Featured Work  

### 🎮 Tic-Tac-Toe Game  
- Classic 2-player Tic-Tac-Toe game on a 3×3 grid.  
- Detects **wins, losses, and draws**.  
- Includes **restart/reset functionality**.  
- Example: `tic-tac-toe/index.html`
